## Running the test

[Link to the task](https://github.com/rolling-scopes-school/basic-nodejs-course/blob/master/descriptions/testing.md)

Ways to run the test:

Running testing:

```
npm test

```

Running detailed testing

```
npm test -- --verbose

```

Running coverage

```
npm run cover

```
