# Инструкция

## Запуск теста

[Ссылка на задание](https://github.com/rolling-scopes-school/basic-nodejs-course/blob/master/descriptions/testing.md)

Способы запуска теста:

Запуск тестирования:

```npm test

```

Запуск подробного тестирования

```npm test -- --verbose

```

Запуск расчета покрытия

```npm run cover

```
